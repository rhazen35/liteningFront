thumbs_up_down Pros and Cons
thumb_up Pros

When working on a project I find it best to take the time and think about business logic and how your view is on the matter. There is little use in rushing the project's core functionality for the sake of a deadline, when later you find important missing pieces that change the core.

This means that when I work on your project I prefer to have continious communication as well as a plan that can be communicated upon. The demo section of this website provides a preview of the work I have done so far on your project.

thumb_down Cons

I'm a Web Developer, not a Web Designer.

A Web Developer builds the backbone of the web solution, using the programming languages ranked above.
A Web Designer uses graphic design software such as Adobe Photoshop, Illustrator and InDesign.

This does not mean I can not build you a good looking web solution, it does mean that I am not the person that will design your logo, nor designing illustrations.
